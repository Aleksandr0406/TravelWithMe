## TravelWithMe

**ScheduleTracker** — iOS- приложение, предназначенное для поиска маршрутов и графиков движения поездов. Пользователи указывают начальные и конечные пункты следования и по завершению запроса формируется перечень доступных рейсов, который можно фильтровать по времени и наличию пересадок. Дополнительно реализована функция просмотра сторис, переходящими в полный экран при открытии

## Возможности

- Выбор направления: города и станции отправления и прибытия
- Переключение «Откуда» ↔ «Куда» одним нажатием
- Поиск и отображение расписания поездов
- Фильтрация по времени суток и пересадкам
- Экран с деталями маршрута (время, дата, продолжительность, перевозчик)
- Сторис в формате карточек, открывающихся на весь экран
  
## Стек

- **Язык**: Swift
- **UI**: SwiftUI
- **Архитектура**: MVVM
- **Данные**: встроенные JSON / моковые данные
- **Минимальная iOS**: 17.6+
